div와 span

가장 많이 사용하는 HTML 태그
display 설명
block 블럭을 하나하나 쌓아올리는 형식의 요소 (div)
여러 블럭을 연속으로 배치하면, 화면에서 세로로 배치된다
block요소는 너비와 높이를 지정할 수 있다 (기본 너비 : 100%)
inline 한 줄 안에서 옆으로 배치되는 형식의 요소 (span)
여러 인라인을 연속으로 배치하면, 화면에서 가로로 배치된다
inline요소는 너비와 높이를 지정할 수 없다 (기본 너비 : 내용만큼만)
