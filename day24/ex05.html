위치, 크기, 간격 주요 style 속성

속성 설명 값 예시
display 요소가 화면에 표시되는 형식을 지정한다 block, inline, inline-block, flex, none
border 요소의 테두리를 지정한다. 두께, 유형, 색상 순으로 지정한다 0, 1px solid black, 2px dashed red, 0.5px double blue;
margin 테두리 바깥쪽 여백을 지정한다. 전체 || 상하 좌우 || top right bottom left 10px, 20px 10px, 20px auto, 5px 10px 5px 7px
padding 테두리 안쪽 여백을 지정한다. 전체 || 상하 좌우 || top right bottom left 10px, 20px 10px, 20px auto, 5px 10px 5px 7px
color 글자의 색상을 지정한다. 색상이름, #rrggbb, rgb(0, 255, 128) red, #ff0000, #f00, rgb(255, 0, 0)
background-color 요소의 배경색을 지정한다. 내용 + 패딩의 크기만큼 적용된다 color 의 값과 동일한 형식
width 요소의 너비를 지정한다. block 의 너비는 기본 100%이다 100px, 100%, 50em, 2rem, 90vw
height 요소의 높이를 지정한다. block은 내용만큼 높이가 지정된다 100px, 100%, 50em, 2rem, 90vh
