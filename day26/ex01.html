input 관련 태그

태그 예시 설명
input type="text" 사용자에게 문자열 기반의 텍스트를 입력받기 위한 태그
type="number" 사용자에게 정수 및 실수 기반의 텍스트를 입력받기 위한 태그
type="password" 사용자에게 문자열 기반의 텍스트를 입력받기 위한 태그
입력받은 내용을 화면상 표시하지 않는다 (가려서 보여준다)
type="email"
스트링 형식의 email 주소를 입력받는다
형식에 맞지 않으면, 제출할 수 없다
type="radio" 미성년자 사용자에게 마우스를 이용하여 항목을 선택한다
이름이 같은 항목은 여러개 중 하나만 선택할 수 있다
type="checkbox" 빨강 초록 파랑 사용자에게 마우스를 이용하여 항목을 선택한다
이름이 같은 항목은 여러개를 중복 선택할 수 있다
type="range" 사용자에게 마우스로 숫자의 범위를 지정하여 입력받는다
입력받는 범위를 지정할 수도 있다
type="date" 사용자에게 키보드 및 마우스로 날짜를 입력받는다
날짜는 yyyy-MM-dd 형식의 텍스트이다
type="time" 사용자에게 키보드 및 마우스로 시간을 입력받는다
날짜는 a hh:mm 형식의 텍스트이다
type="file" 사용자에게 첨부 파일을 입력받는다
파일은 문자열 기반이 아니므로, 별도의 규칙이 필요하다
type="hidden" 화면에 나타나지 않는 유형의 input 값을 나타낸다
input의 type이 hidden이면 display속성이 none이다(자리차지 없음)
type="submit" from 태그 내부의 input 및 입력 요소의 값을 action에 전달한다
button 태그가 form 태그 내부에 있다면 동일한 역할을 수행한다
type="button" 외형 상 일반 button 및 submit과 동일하지만 아무런 작동이 없다
주로, javascript를 이용하여 from 내부에서 이벤트를 부여하기 위해 사용한다
select 여러 목록 중 하나를 선택하여 값을 지정한다
기능 상 input type="radio"와 유사하다
textarea 여러 줄에 걸쳐서 입력받는 필드를 생성한다
pre 태그와 마찬가지로 , 엔터를 줄바꿈으로 식별한다
