기초적인 css 선택자

형식 예시 설명
태그 선택자 
div {
    margin: 0;
}
지정한 태그에 해당하는 모든 요소를 선택한다
클래스 선택자 
.item {
    padding: 10px;
}
지정한 클래스에 해당하는 모든 요소를 선택한다
id 선택자 
.logo {
    font-weight : bold;
    text-align : center;
}
지정한 id에 해당하는 모든 요소를 선택한다
자손 선택자 (A > B) 
ul > li {
    color : blue;
}
A 바로 하위에 있는 B들을 선택한다
후손 선택자 (A B) 
#nav .item {
    width : 100px;
}
A 하위에 있는 B들을 선택한다
(포함만 되어있으면 단계에 상관없이 적용)
가상 선택자 (:) 
a:hover {
    text-decoration : underline;
}
지정한 요소가 특정 조건을 만족할 떄,
스타일을 적용한다
